# Start a command prompt as an administrator
1. Press Windows key
2. Type `cmd`
3. Ctrl + Shift + Enter

# Note
* `start` command regards the first double-quoted string as a title.
  * https://technet.microsoft.com/en-us/library/cc770297.aspx
